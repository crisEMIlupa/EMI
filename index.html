ELEMENTOS DE HTM5

ElementoDescripciónEjemplo
time Representa un valor de fecha y hora
articleDefine contenido autónomo que podría existir independientemente del resto del contenido.
hearderDefine la cabecera de una página o sección. Usualmente contiene un logotipo, el título del sitio Web y una tabla de navegación de contenidos.
mainDefine el contenido principal o importante en el documento. Solamente existe un elemento
en el documento.
colorPermite elegir el color
passwordSe usa para poner la contraseña
audioRepresenta un sonido o stream de audio. audio src= "la direccion de a musica autoplay>"
videoRepresenta un video , y sus archivos de audio y capciones asociadas, con la interfaz necesaria para reproducirlos. video src="videofile.ogg" autoplay poster="posterimage.jpg">
sourcePermite a autores especificar recursos multimedia alternativos para los elementos multimedia como video o audio.
outputRepresenta el resultado de un cálculo.
progressRepresenta el progreso de finalización de una tarea.
commandRepresenta un comando que un usuario puede invocar.

TIPOS DE META TAGS

ElementoDescripciónEjemplo
Meta DescriptionLa descripción meta de HTML se usa para dar un breve resumen de la página web que también es útil para los motores de búsqueda.meta name="description" content="Poori Padhai provides you free tutorial of HTML,CSS,JS etc.">
Meta KeywordHTML Meta Keyword se utiliza para definir las palabras clave para los motores de búsqueda.meta name="keywords" content="HTML,CSS,Javascript">
Meta AuthorMeta Author se utiliza para dar al autor de la página web.meta name="author" content="Anurag Gupta">
Meta RefreshMeta Refresh se usa para actualizar la página web en cada intervalo de tiempo en particular.meta http-equiv="refresh" content="5">
